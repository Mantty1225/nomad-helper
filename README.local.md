# 🌍 数字游民签证助手

AI帮你找到最适合的数字游民目的地

## 🚀 快速开始

### 本地运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```

### 在线访问
- **Render**: https://nomad-helper.onrender.com
- **Streamlit Cloud**: https://nomad-helper.streamlit.app

## 📝 使用方法

1. 在左侧填写你的信息
   - 职业
   - 月收入（人民币）
   - 目标大洲
   - 偏好（可多选）
   - 家庭情况

2. 点击"获取推荐"按钮

3. 查看右侧的个性化推荐

## 🎯 功能特点

- 智能匹配算法，根据你的情况推荐最适合的国家
- 详细的国家信息：优势、劣势、收入要求、生活成本
- 直观的匹配度评分
- 支持多种职业和家庭情况

## 🇬🇧 支持的国家

- 🇵🇹 葡萄牙
- 🇪🇸 西班牙
- 🇹🇭 泰国
- 🇲🇽 墨西哥
- 🇭🇷 克罗地亚

## 📊 匹配因素

系统根据以下因素进行智能匹配：

- 收入水平
- 职业类型
- 目标大洲
- 个人偏好
- 家庭情况

## 🔧 技术栈

- **Streamlit**: Web界面
- **Python**: 后端逻辑
- **Pandas**: 数据处理
